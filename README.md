# Meme Project

A small practice project from a Udemy web development course focused on creating and styling a motivational meme webpage.

## New Concepts Practiced

- Using the CSS box model
- Working with percentage-based widths and margins
- Centering a container with CSS
- Importing and applying Google Fonts
- Styling images with borders and responsive widths
- Aligning text with `text-align`
- Transforming text to uppercase
- Setting background and text colors

## Technologies Used

- HTML
- CSS
